# CartToDoorstep-E-commerce-Web-Application

Project Title

CartToDoorstep – E-commerce Platform

Brief Description

CartToDoorstep is a web-based e-commerce platform that allows customers to browse products, place orders, and track deliveries, while sellers manage inventory and orders through a dedicated dashboard. The system also includes an admin panel to manage users, sellers, and product listings.

Key Features

Customer Module

Browse products and categories

Add products to cart and place orders

View order history and order status

Seller Module

Add, update, and manage product listings

View and process customer orders

Manage product inventory and pricing

Admin Module

Manage sellers Monitor orders and platform activity

Control product approvals and categories

Technologies Used

Frontend: React, HTML, Boostrap, JavaScript

Backend: Express.js

Database: MongoDB
